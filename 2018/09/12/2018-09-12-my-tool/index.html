推荐我Mac上经常使用的30款软件

  • Larry
  • 2018-09-12
  • 有故事的开发者
  • 次

工欲善其事，必先利其器，苹果发布会即将来临，今天把我平时使用频率比较高的软件做一个汇总，不妨先看一看我的利器，也许对你有所帮助。

1. Xcode 10.0 beta

吃饭工具，iOS开发必备，目前使用的是最新beta版，支持暗黑色主题，使用舒服，一般是使用beta版开发，正式版提交构建版本，Android开发工具Android Studio就不说了。

2. Google Chrome Canary

Google推出的一个特别版Chrome，都称它为金丝雀版，其实就是开发者版本，使用一段时间后我觉得它的加载页面速度更快。

3. iTerm2

绝对是Mac OS的终端利器，之前我一直使用的是Mac OS自带的终端，但系统自带的主题太有限，所以发现这款软件，不仅强大而且支持主题配置，主题列表：https://github.com/robbyrussell/oh-my-zsh/wiki/themes，你可以挑选自己喜爱的主题进行配置。

4. VSCode

VS Code 全称 Visual Studio Code，是微软推出的一款免费的、开源的、高性能的、跨平台的、轻量级的代码编辑器，号称微软最好的开源软件作品。

VS Code有一个扩展和主题市场，包含了数以千计质量不同的插件，它提供了 Windows、macOS 和 Linux版本，内置支持JavaScript、TypeScript 和 Node.js，通过丰富的扩展和插件生态，你可以使用 VS Code 高效的编写 C++、C#、Java、Python、PHP、Go 等主流编程语言。

在此之前我用Sublime Text，现在我用VS Code。

5. SourceTree

免费的Git图形化管理工具、项目版本控制软件，适用于git项目管理，使用它管理代码，基本够用。

6. Sequel Pro

Mac上的MySQL管理工具，是我经常使用的工具，它支持包括索引在内的所有表管理功能，支持MySQL视图，可以同时使用多个窗口来操作多个数据库/表。

7. ForkLift

远程/本地文件管理利器，我更看重它的本地文件管理器，很强大，平时配合FileZilla使用。

8. Eagle

平时自己设计的内容太多，正好用它解决大量图片素材「收藏、整理、查找」的各种困扰,让你可以轻松管理各种图片，提升工作效率。

9. Notion

是一款「将笔记、知识库和任务管理无缝整合的协作平台」，它是知识库、任务管理的神器，平时使用它轻松管理自己的App开发进度和任务。

10. bear

颜值即正义，它是一款可以快速提升你写作质量与效率的神奇工具，在设计上精致与易用，功能上强大却不失便捷，满足了我对「记笔记」的设想和需求。

11. MacDown

它是一款MarkDown编辑器工具，提到markdown工具市面上不胜枚举，使用它的主要原因在于它的轻量、简洁。

12. OmniGraffle

它是一款功能强大的绘图软件。可以用来绘制图表，流程图，组织结构图以及插图，也可以用来组织头脑中思考的信息，组织头脑风暴的结果，绘制心智图，作为样式管理器，或设计网页或PDF文档的原型，我使用它的主要用途是绘制流程图。

13. Things

Things是去年开始使用的，设计简洁大方，交互方式自然流畅，用它来管理自己的待办事项甚是舒心。

14. MAMP

它是一款很棒的在MAC下运行的网站集成环境软件，功能强大，配置简单，PHP本地调试我一般都是使用它。

15. Postman

它是一个很强大的API调试、Http请求的工具，允许用户发送任何类型的 HTTP 请求，例如 GET、POST、HEAD、PUT、DELETE等。

16. Magnet

它是一款Mac上优秀的窗口大小控制工具，每当你想将内容从一个应用程序复制到另一个应用程序、并排比较文件或任何其它方式的多任务时，你需要所有窗口都能相应地排列。Magnet 让这一流程变得整洁便利。你只需一次拖动窗口到边沿，就能将任何窗口捕捉到你的左、右、上或下半个屏幕中。

17. iHosts

这也是我经常使用的软件之一，如果你需要修改 /etc/hosts 来切换开发环境，甚至已经在用脚本、工具来编辑 Hosts ，你真的需要试试 iHosts。

18. 小历

小历是 macOS & iOS 平台上小而美的日历，macOS 上通过菜单栏和通知中心让用户查看日历十分便捷，我也是近期才发现，很好用。

19. pap.er

自从换了新Mac，一直在找一款桌面壁纸软件，直到发现了pap.er，小而精，每天享受来自全球新鲜精美的壁纸。

20. Easy New File

它是一款小巧的 Finder 扩展应用，可以为 Finder 右键菜单中添加「新建文件」、「拷贝路径」、「打开终端」等功能选项，其中「新建文件」一项可支持许多格式文件的创建操作，比如纯文本、富文本、Word/Excel/PPT、Markdown、HTML、XML 以及各类编程脚本，我也是最近才使用，觉得不错。

21. 幕布

极简大纲笔记 | 一键生成思维导图，它是一款超好用的思维导图辅助工具，大纲笔记与思维导图可以一键转换。

22. iMazing

iMazing是一款比iTunes 好用很多很多的iOS设备管理软件，你可以用来进行备份数据、备份应用（应用数据）、传输文件、数据迁移，甚至可以备份应用，就算下架也能再次安装。

23. Sip

Mac上一款清爽简单的颜色选择工具，它可以帮助你提取任意颜色，屏幕上的任何地方都可以获取像素颜色编码。

24. Paste

Mac平台上简洁好用的剪贴板工具，软件拥有如原生般美观的界面和各种好用的功能，足够应付你日常对于剪贴板工具的各种需求。

能够自动识别你复制的内容格式，包括文本、图像、链接、文件甚至是其它任何类型的格式，软件都可以帮你保留它们。

25. IINA

最有可能变得完美的macOS「视频播放器」，它是一款开源的视频播放器，支持 Touch Bar、兼容 MPV 脚本、几乎支持所有格式、网络播放、拥有优雅外观的视频播放器。

26. GIF Brewery 3

是Mac上超好用的动图制作应用，支持从视频中截取片段制作动图，还支持调节长宽尺寸，制作出来的动图体积小画质高。

GIF Brewery 3 可以通过导入视频、从摄像头拍摄视频、桌面屏幕录制、网络视频链接四种方式获得需要制作 GIF 的视频源。

27. Telegram Desktop

是一个跨平台的实时通讯应用，它大概是国外最有逼格的聊天软件，专为聊天而生，喜欢暗黑的主题。

28. 微信多开

WeChatTweak 是一款微信客户端插件，仅支持 macOS，可实现阻止消息撤回、消息列表通知、系统通知、正常撤回自己发出的消息、客户端无限多开，具体的实现教程可自行google查看，地址：https://github.com/Sunnyyoung/WeChatTweak-macOS。

另外一种方法使用weweChat，下载地址：https://github.com/trazyn/weweChat

29. HapticKey

可以为你的Macbook Pro 带来 Touch Bar 触控反馈的工具，HapticKey 运行后默认不会自动启用，你需要去开启它的反馈功能，除此以外，你还可以使用音效和屏幕闪烁反馈，我使用后觉得效果不好，音效和屏幕闪烁还是别启用了。

30. 其它设计软件

我Mac上设计相关的软件全部聚集在这里，经常使用的是PS、AI、Sketch，其它平时偶尔使用。

今天先说到这，凌晨一点就是苹果发布会了，也许你又要开始剁手了，收下这波利器，为你的新Mac提升效率，我的汇总可能也有遗漏，如果你有好的利器，欢迎扫码关注码力全开留言推荐给大家~
