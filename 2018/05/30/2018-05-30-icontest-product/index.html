Icon Test — 手机上快速预览应用图标的神器

  • Larry
  • 2018-05-30
  • 个人作品
  • 次

好的设计不仅仅是为了审美。这是关于你付出多少汗水的细节。

为了创建一个满意的图标，你最终必须测试它的许多版本在真实设备上看看哪一个是最好的。

不知你是否有过这样的烦恼，产品icon不断调整，为了更直观的让大家看到真实的效果，你不得不导出不同版本、组合的图标给大家，事儿不难但却繁琐。

今天给大家介绍的App —— Icon Test，就是这样一款可以帮你快速预览应用图标的工具。

Icon Test的来源

前段时间再确定码力全开logo的时候，由于一直再不断调整，前后画了十几个，所以想把这些logo以应用图标的形式展示到手机上查看效果，想到了以前用过的ICNS，ICNS的思路其实很简单，设计好logo，导出到dropbox文件夹，打开应用预览，但目前此款应用找不到了。

就这样开始了Icon Test的开发。

Icon Test 有哪些功能？

Icon Test首页类似iOS原生系统的宫格布局，你只需要把做好的Icon 发到手机上，打开App随意点击任何一个图标即可替换，同时点击名字也可以修改。

如果你想看一看修改后的logo在不同背景不同位置下的效果，又或者你觉得它不太满意，想删除，删除了又想恢复？

Icon Test都可以满足你的需求，它不仅提供了自定义背景和官方背景图，同时还具有模糊背景突出logo的功能，logo支持长按移动和删除，删除后的Icon会自动收纳到回收站，长按即可恢复。

经过一番调整之后，终于确定了最终的logo，想保存分享？又或者分享之前想用画笔圈出你最终确定的那个，这些功能Icon Test同样具备。


确定了logo，但App开发往往需要不同尺寸的logo，譬如：

  1. 开发者平台 - QQ、微信、微博等.
  2. Xcode工程AppIcon.

注：导出之后的不同尺寸logo是jpg格式。

最后

Icon Test作为一款刚上架的App，还存在很多需要优化的问题，希望版本持续的迭代能给你们带来越来越好的功能和体验。

Icon Test目前AppStore定价3元，有需要的小伙伴可以去下载体验。
